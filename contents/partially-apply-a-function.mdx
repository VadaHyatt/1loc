title: Partially apply a function
---

**JavaScript version**

```js partial.js
const partial =
    (fn, ...a) =>
    (...b) =>
        fn(...a, ...b);
```

**Examples**

```js examples.js
const sum = (x, y) => x + y;
const inc = partial(sum, 1);
inc(9); // 10
```

**JavaScript version**

```js radsToDegs.js
const radsToDegs = (rad) => (rad * 180) / Math.PI;
```

**TypeScript version**

```ts radsToDegs.ts
const radsToDegs = (rad: number): number => (rad * 180) / Math.PI;
```

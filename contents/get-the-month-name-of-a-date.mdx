title: Get the month name of a date
updated: '2021-10-13'
---

**JavaScript version**

```js getMonthName.js
// `date` is a Date object
const getMonthName = (date) =>
    [
        'January',
        'February',
        'March',
        'April',
        'May',
        'June',
        'July',
        'August',
        'September',
        'October',
        ' November',
        'December',
    ][date.getMonth()];
```

**TypeScript version**


title: Calculate the sum of arguments
updated: '2021-10-13'

**JavaScript version**

```js sum.js
const sum = (...args) => args.reduce((a, b) => a + b);
```

**TypeScript version**

```ts sum.ts
const sum = (...args: number[]): number => args.reduce((a, b) => a + b);
```

**Examples**

```js examples.js
sum(1, 2, 3, 4); // 10
```

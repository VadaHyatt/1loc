**JavaScript version**

```js mul.js
const mul = (...args) => args.reduce((a, b) => a * b);
```

**TypeScript version**

```ts mul.ts
const mul = (...args: number[]): number => args.reduce((a, b) => a * b);
```

**Examples**

```js examples.js
mul(1, 2, 3, 4); // 24
```

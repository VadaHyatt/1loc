**JavaScript version**

```js plusOneYear.js
const plusOneYear = ((d) => new Date(d.setFullYear(d.getFullYear() + 1)))(new Date());
```

**TypeScript version**

```ts plusOneYear.ts
const plusOneYear: Date = ((d) => new Date(d.setFullYear(d.getFullYear() + 1)))(new Date());
```

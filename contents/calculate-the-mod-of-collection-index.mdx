**JavaScript version**

```js mod.js
const mod = (a, b) => ((a % b) + b) % b;
```

**TypeScript version**

```ts mod.ts
const mod = (a: number, b: number): number => ((a % b) + b) % b;
```

**Examples**

```js examples.js
mod(-1, 5); // 4
mod(3, 5); // 3
mod(6, 5); // 1
```

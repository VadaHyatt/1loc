updated: '2021-10-22'

**JavaScript version**

```js union.js
const union = (...arr) => [...new Set(arr.flat())];
```

**TypeScript version**

```ts union.ts
const union = <T,_>(...arr: T[][]): T[] => [...new Set(arr.flat())];
```

**Example**

```js example.js
union([1, 2], [2, 3], [3]); // [1, 2, 3]
```

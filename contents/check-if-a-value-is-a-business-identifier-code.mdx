title: Check if a value is a business identifier code
updated: '2021-10-13'
---

**JavaScript version**

```js isBIC.js
const isBIC = (value) => /^[a-zA-Z]{6}[a-zA-Z0-9]{2}([a-zA-Z0-9]{3})?$/.test(value);
```

**TypeScript version**

```ts isBIC.ts
const isBIC = (value: string): boolean => /^[a-zA-Z]{6}[a-zA-Z0-9]{2}([a-zA-Z0-9]{3})?$/.test(value);
```

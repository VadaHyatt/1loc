**JavaScript version**

```js degsToRads.js
const degsToRads = (deg) => (deg * Math.PI) / 180.0;
```

**TypeScript version**

```ts degsToRads.ts
const degsToRads = (deg: number): number => (deg * Math.PI) / 180.0;
```

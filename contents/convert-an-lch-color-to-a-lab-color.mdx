**JavaScript version**

```js lchToLab.js
const lchToLab = (l, c, h) => [
    l,
    Math.cos(h * Math.PI / 180) * c,
    Math.sin(h * Math.PI / 180) * c,
];
```

**TypeScript version**

```ts lchToLab.ts
const lchToLab = (l: number, c: number, h: number): number[] => [
    l,
    Math.cos(h * Math.PI / 180) * c,
    Math.sin(h * Math.PI / 180) * c,
];
```

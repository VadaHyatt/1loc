title: Check if a date is a weekday
updated: '2021-10-13'
---

**JavaScript version**

```js isWeekday.js
// `date` is a Date object
const isWeekday = (date = new Date()) => date.getDay() % 6 !== 0;
```

**TypeScript version**

```ts isWeekday.ts
const isWeekday = (date = new Date()): boolean => date.getDay() % 6 !== 0;
```

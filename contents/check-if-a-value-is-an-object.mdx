title: Check if a value is an object
updated: '2021-10-13'
---

**JavaScript version**

```js isObject.js
const isObject = (v) => v !== null && typeof v === 'object';
```

**TypeScript version**

```ts isObject.ts
const isObject = (v: any): boolean => v !== null && typeof v === 'object';
```

**Examples**

```js examples.js
isObject(null); // false
isObject('hello world'); // false

isObject({}); // true
isObject([]); // true
```
